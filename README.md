# Selenium Java Framework
This is a Selenium Java Framework for automated testing of web applications. It provides a structured approach to writing and executing test cases using Selenium WebDriver.

## Features
- Page Object Model (POM) design pattern for better maintainability and readability.
- TestNG framework for test execution and reporting.
- Maven for dependency management.
- Cross-browser testing support.
